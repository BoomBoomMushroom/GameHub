Account Search

Url: https://gameshub-api.herokuapp.com/accsearch

Method: GET

Args: q

q: input query for search

Returns html page of accounts that usernams start with the query param


Get Account

Url: https://gameshub-api.herokuapp.com/getaccount

Method: GET

Args: username

username: username input

Returns json of user's account


Logout

Url: https://gameshub-api.herokuapp.com/logout

Method: POST

Args: token

token: account token to verify that you are in the account

Returns status code


Login

Url: https://gameshub-api.herokuapp.com/login

Method: GET

Args: username, password

username: username input, password: password input

Returns token if successful else returns INVALID_ACCOUNT_TOKEN


Delete Account

Url: https://gameshub-api.herokuapp.com/delacc

Method: POST

Args: token

token: token to verify account

Return status code


Signup

Url: https://gameshub-api.herokuapp.com/signup

Method: POST

Args: username, password

username: username to signup, password: password to signup with

Return status code aslong as if the username isn't taken and is within the correct length and password is within correct length
